## 📘 Experiment Descriptions

### 1️⃣ A* Search Algorithm for Pacman Path Finding
This experiment implements the **A\* search algorithm** to find the optimal path for Pacman to reach its destination in a grid-based environment.  
The algorithm uses heuristic functions along with cost calculations (`f = g + h`) to efficiently explore paths while avoiding obstacles.

**Key Concepts:**  
- Heuristic search  
- Optimal path finding  
- Open and closed lists  

---

### 2️⃣ Minimax Algorithm for Tic-Tac-Toe
This experiment demonstrates the **Minimax algorithm** by building an AI-powered Tic-Tac-Toe game.  
The AI evaluates all possible moves recursively to choose the optimal strategy, ensuring it never loses.

**Key Concepts:**  
- Game trees  
- Decision making  
- Adversarial search  

---

### 3️⃣ Question Answering System using Reasoning Techniques
A **rule-based and similarity-based chatbot** is developed using Natural Language Processing (NLP).  
The system processes user queries using TF-IDF and cosine similarity to retrieve the most relevant responses.

**Key Concepts:**  
- NLP preprocessing  
- TF-IDF  
- Cosine similarity  
- Reasoning systems  

---

### 4️⃣ Handwritten Digit Recognition using CNN
This experiment implements a **Convolutional Neural Network (CNN)** using Keras to recognize handwritten digits from the MNIST dataset.  
The model is trained and evaluated to measure accuracy and loss.

**Key Concepts:**  
- Deep learning  
- Convolutional Neural Networks  
- Image classification  

---

### 5️⃣ Face Recognition using Open-Source Frameworks
A face recognition system is developed using the **face_recognition** library.  
The program encodes known faces and compares them with unknown images to identify matches.

**Key Concepts:**  
- Feature extraction  
- Face encoding  
- Pattern matching  

---

### 6️⃣ Object Detection Model for Robot Arm Tracking
This experiment uses **TensorFlow and pre-trained deep learning models** to detect objects in images.  
Bounding boxes and labels are drawn around detected objects, simulating object recognition for robotic applications.

**Key Concepts:**  
- Object detection  
- Deep learning frameworks  
- Pre-trained models
